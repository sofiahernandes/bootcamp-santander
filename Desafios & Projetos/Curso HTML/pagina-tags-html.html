Tags Principais (Estrutura Básica) ↑

O HTML é uma Linguagem de Marcação, ou seja, é um conjunto de tags (marcações), as quais exercem funções estruturais dentro das Aplicações Web. Nesse contexto, algumas dessas tags se destacam por sua alta taxa de utilização. Entre estas, estão:

  • <h1> a <h6> – Cabeçalhos de diferentes níveis, do mais importante (<h1>) ao menos importante (<h6>).
  • <p> – Define um parágrafo de texto.
  • <strong> – Dá ênfase forte ao texto (geralmente em negrito).
  • <i> – Aplica estilo itálico ao texto.
  • <u> – Aplica sublinhado ao texto.
  • <small> – Exibe o texto em tamanho reduzido.
  • <mark> – Destaca o texto com cor de fundo (geralmente amarelo).
  • <abbr> – Define uma abreviação com significado. Por exemplo:
    <abbr title="World Health Organization">WHO</abbr>
  • <ol> – Cria uma lista ordenada (numerada).
  • <ul> – Cria uma lista não ordenada (com marcadores).
  • <li> – Define um item dentro de listas (<ol> ou <ul>).
  • <a> – Define um link:
    <a href="http://dio.me">Clique aqui</a>
  • <hr> – Insere uma linha horizontal como divisor de seção.
  • <sub> – Texto em subscrito (abaixo da linha).
  • <sup> – Texto em sobrescrito (acima da linha).
  • <blockquote> – Define uma citação em bloco (longa).
  • <font> – Define fonte, cor e tamanho do texto (obsoleta/"deprecated").
  • <del> – Texto "removido" (riscado).

Estruturas de dados (Forms, Listas e Tabelas) ↑

O HTML também permite a criação de estruturas de dados mais complexas, como formulários, listas e tabelas. Essas estruturas são essenciais para a interação do usuário com a aplicação web.

  • Formulários: Utilizados para coletar dados do usuário. A tag principal é <form>, que pode conter campos de entrada (<input>), áreas de texto (<textarea>), botões (<button>) e outros elementos interativos.
  • Listas: Podem ser ordenadas (<ol>) ou não ordenadas (<ul>), com itens definidos por <li>, como mostrado na lista de tags acima. As listas são úteis para organizar informações de forma clara.
  • Tabelas: Criadas com a tag <table>, contendo linhas (<tr>) e células (<td>). As tabelas são usadas para exibir dados tabulares de maneira estruturada.

HTML Semantico e Acessibilidade ↑

O HTML semântico é uma abordagem que utiliza tags com significado claro e específico, melhorando a compreensão do conteúdo por navegadores e leitores de tela. Isso é crucial para a acessibilidade, permitindo que pessoas com deficiências visuais ou cognitivas acessem o conteúdo de forma mais eficiente.

Algumas tags semânticas importantes incluem:

  • <header> – Define o cabeçalho de uma página ou seção.
  • <nav> – Define uma seção de navegação.
  • <main> – Representa o conteúdo principal da página.
  • <article> – Define um conteúdo independente e auto-contido.
  • <section> – Agrupa conteúdo relacionado dentro de uma página.
  • <aside> – Define conteúdo relacionado, mas não essencial ao conteúdo principal.
  • <footer> – Define o rodapé de uma página ou seção.
